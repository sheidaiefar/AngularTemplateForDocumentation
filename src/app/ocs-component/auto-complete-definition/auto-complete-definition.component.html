How to Use of OCS AutoComplete Component

 

The ocs-auto-complete  provides proper ability for working with series of data and loading related titles as its been searching by user ,it will be very usefull when we have a huge amount of data.

In order to use an autoComplete component, you must perform the following steps:

  1. Import the specific Autocomplete module "OcsAutocompleteModule" into your current module which has been imported from the "ocs-ui-core" ibrary. 

  2. Insert its specific tag into the current template :  <ocs-autocomplete></ocs-autocomplete>


  3. This tag could have these attributes values:

 

  •   size: InputControlSize;

  •   alignment: TextAlignment;

  •   required: boolean;

  •   hint: string;

  •   placeholder: string;

  •   label: string;

  •   showMessages: boolean;

  •   readonly: boolean;

  •   suggestions: number;

  •   dataProvider: DataProviderFunc<any>;

  •   showValidationIcon: boolean;

  •   itemValue: string;

  •   itemText: string;

  •   optionSelect: EventEmitter<any>;
