How to Use of OCS Dropdown's Component

 

Dropdowns are toggleable, contextual overlays for displaying lists of links and more.

The ocs-drop-down component provides useful utilities for this purpose.

A Dropdown Component looks like :

Users {{us.fullName}}

 

In order to use a ocs-drop-down component, you must perform the following steps:

  1. Import the specific dropdown module "OcsDropDownModule" into your current module which has been imported from the "ocs-ui-core" ibrary. 

  2. Insert its specific tag into the current template : <ocs-drop-down></ocs-drop-down>

  3. This tag could have these attributes values:

 

  •     required: boolean;

  •      size: InputControlSize;

  •     alignment: TextAlignment; 

  •      required: boolean;
  •     hint: string;
  •     placeholder: string;
  •     label: string;
  •     showMessages: boolean;
  •     readonly: boolean;
  •     items: any;
  •     itemValue: string;
  •     itemText: string;
  •     itemsPreset: DropDownItemsPreset;
  •     selectionChange: EventEmitter<any>;
  •     _addons: QueryList<OcsDropDownAddonDirective>;
  •     _inputBase: OcsInputBaseComponent;
  •     _host: NgModel;
  •     _selectedItem: any;
  •     _value: any;
  •     _stdBag: StandardStringBag;
     

 

To use of dropdown component its neededto import "OcsDropDownModule" in the current module which has been imported from the

"ocs-ui-core"

library.

To demonstrate a dropdown which toggles between some users(shows fullname of users) and returns an id as the selected user ; must set these attributes:


<ocs-drop-down [(ngModel)]="selectedUserId" name="selectedUser" [label]="Users" [items]="UsersArray" itemText="fullname" itemValue="id"></ocs-drop-down>

 

  • selectedUserId : is a variable which returns the id of the selected user

  • items : represents an array of user model 
  • itemText : refers to the field of fullname in user model and shows in the labels of the dropdown
  • itemValue : refers to the field of id in user model
