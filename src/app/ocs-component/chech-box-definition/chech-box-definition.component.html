How to Use of OCS CheckBox Component

 

The ocs-check-box allows the user to select one or more items from a set.

A Basic CheckBox Component,created by ocs component looks like :


In order to use the CheckBox component, you must perform the following steps:

    1. Import the specific CheckBox module "OcsCheckBoxModule" into your current module which has been imported from the "ocs-ui-core" ibrary. 

    2. Insert its specific tag into the current template :  <ocs-checkbox></ocs-checkbox>
    3. This tag could have these attributes values:
      • label: String;
      • checked: boolean;
      •  checkedChange: EventEmitter<boolean>;
