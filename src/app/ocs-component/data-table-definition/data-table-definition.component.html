How to Use of OCS DataTable Component

 

Tables display information in a way that’s easy to scan, so that users can look for patterns and insights. They can be embedded in primary content, such as cards. A data table contains a header row at the top that lists column names, followed by rows for data.

The ocs-data-table package provides useful utilities for working with data sets;

A Basic Data Table Component,whci can be created by ocs component looks like:

{{'Name'}} {{row.name}} {{'Age'}} {{row.age}}

In order to use an OcsDataTable component, you must perform the following steps:

  1. Import the specific OcsDataTable module "OcsDataTableModule " into your current module which has been imported from the "ocs-ui-core" ibrary. 

  2. Insert its specific tag into the current template :  <ocs-data-table></ocs-data-table>
    This tag could have these attributes values:

  •   dataProvider: DataProviderFunc<any>;

  •   dataItemKey: string;

  •   size: DataGridControlSize;

  •   hoverable: boolean;

  •   pageable: boolean;

  •   pageSize: number;

  •   pageIndex: number;

  •   autoLoad: boolean;

  •   filterable: boolean;

  •   sortable: boolean;

  •   selectable: boolean;

  •   selectionMode: DataGridSelectionMode;

  •   showHeader: boolean;

  •   rowCssClass: any;

  • selectedItems: Array<any>;

  • selectedItemsChange: EventEmitter<any[]>;

  • pageSizeOptions: number[];

  • showRowDetails: boolean;

  • showRowNumbers: boolean;
