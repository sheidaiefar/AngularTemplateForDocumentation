How to Use of OCS Button's Component

 

The ocs-number-box package provides useful utilities for working with digits input fields ;

A Basic Number Box Component,whci can be created by ocs component looks like :

 

In order to use a number-box component, you must perform the following steps:

  1. Import the specific number-box module "OcsNumberBoxModule" into your current module which has been imported from the "ocs-ui-core" ibrary. 

  2. Insert its specific tag into the current template : <ocs-number-box></ocs-number-box>

  3. This tag could have these attributes values:

 

  •     size: InputControlSize;

  •     required:  boolean;

  •     hint: string;

  •     placeholder: string;

  •     showMessages: boolean;

  •     readonly: boolean;

  •     label: string;

  •     maxlength: number | string;

  •     min: number | string;

  •     max: number | string;

  •     _addons: QueryList<OcsTextBoxAddonDirective>;

  •     _inputBase: OcsInputBaseComponent;

  •     _host: NgModel;

  •    _disabled: boolean;

  •     _error: any;

  •     _value: any;
