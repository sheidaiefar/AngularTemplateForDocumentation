How to Use of OCS Button Component

A basic  ocs-button  component looks like:

In order to use a button component, you must perform the following steps:

  1. the specific button module "OcsButtonModule" into your current module which has been imported from the

    "ocs-ui-core"

    library.



  2. Insert its specific tag into the current template : <ocs-button></ocs-button>

  3. This tag could have these attributes values:
    • type: 'button' | 'submit' | 'reset';
    • size: 'small' | 'normal' | 'large';
    • disabled: boolean;
    • appearance: 'normal' | 'outline' | 'link';
    • color: 'primary' | 'secondary' | 'success' | 'danger' | 'warning' | 'info' | 'light' | 'dark';
    • text: string;
    • icon: string;

  • To demonstrate a button which has a title of "save" in Persian, in addition to a tick icon, and shows a message by clicking on it;
  1. Import "OcsButtonModule" in the current module,

  2. Insert <ocs-button></ocs-button>

  3. Use this attribute for its title : [text]="stdBag.save" 

  4. Inject this code into the constructor's class : 
    @Inject(STANDARD_STRING_BAG) public stdBag: StandardStringBag,

  5. Import related-modules and services from the libraries which are :
    import "StandardStringBag, STANDARD_STRING_BAG " from 'ocs-ui-core';
    import " Inject " from '@angular/core';
     
  6. Insert its icon which you have found from the Font Awesome package ( https://fontawesome.com/): icon="fas fa-check"
  7. Set up its color: color="secondary"

  8. Define its method by an event binding that implements a click handler : (click)="showMessage()"
     
  9. To show the message on the form, inject "UiUtilsService" service into the current class; and use of "showSnackBar" method on it.

  10. Remember to Do Not use any text between the tags!

  11. The final tag would be like this : 

  12.   <ocs-button [text]="stdBag.save" icon="done" class=" mr-2" color="primary"   (click)="showMessage()"></ocs-button>
